Colores en CSS

Model Description Sintax
Usando nombres Podemos especificar un color usando un nombre predefinido color: red;
Usando el modelo RGB Podemos establecer el valor de un color mezclando diferentes intensidades de los tres colores primarios (0 es transparente, 1 no transparente). rgb(red, green, blue)
Usando el valor RGB hexadecimal Conlleva un # por delante seguido de 6 digitos #c0c0c0 /*silver*/
Usando el modelo HSL Utilizamos conceptos mas claros hsl(hue, saturation, lightness) hsl(0, 100%,50%)/* red */

Selectores CSS

Tipo Selector Description Ejemplo
Selector Universal Permiten seleccionar elementos del documento a los que queremos aplicar estilos *{ color: green; }
SELECTORES DE TIPO (p) Permite seleccionar elementos usando el nombre del elemento p{ color: red;}
SELECTOR DE CLASE (.foo) Permite seleccionar elementos en función de los valores del atributo class de los elementos. Para que los elementos sean seleccionados, su atributo class debe incluir un valor que coincida con el valor del selector p class=”foo bar”> ¡HOLA MUNDO! p class=”foo baz”> ¡ADIÓS MUNDO! p
SELECTORES DE ID (#foo) Permite seleccionar un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, su atributo id debe coincidir exactamente con el valor dado en el selector. p id=”foo”> ¡HOLA MUNDO! #foo{color: blue;}
SELECTORES DE ATRIBUTO (a[foo]) Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo. a[title]{color: blue;}

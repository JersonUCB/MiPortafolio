ELEMENTOS HTML

Elemento Descripcion Documentacion
html Representa a la raiz del documento HTML documentacion
div El elemento de división de contenido documentacion
a El elemento ancla, crea un hipervínculo a páginas web documentacion
audio El elemento audio incrustado,se utiliza para incrustar contenido de sonido en los documentos. Puede contener una o más fuentes de audio documentacion
b El elemento traer atencion, se utiliza para llamar la atención del lector sobre el contenido del elemento documentacion
br El elemento Salto de línea, útil para escribir un poema o un discurso, donde la división de líneas es significativa documentacion
cite El elemento cita,se usa para describir una referencia a un trabajo creativo citado y debe incluir el título de ese trabajo documentacion
code El elemento de codigo en linea, muestra su contenido con estilo para indicar que el texto es un fragmento corto de código de computadora documentacion
data El elemento de datos, vincula una determinada pieza de contenido con una traducción legible por máquina documentacion
em El elemento enfasis, El elemento se puede anidar, y cada nivel de anidamiento indica un mayor grado de énfasis documentacion
img El elemento de inserción de imagen, incrusta una imagen en el documento documentacion
kbd El elemento de entrada de teclado, representa un tramo de texto en línea que indica la entrada de texto del usuario desde un teclado, entrada de voz o cualquier otro dispositivo de entrada de texto documentacion
mark El elemento marcar texto, representa texto que está marcado o resaltado para fines de referencia o notación documentacion
s El elemento tachado,representa el texto con un tachado o una línea que lo atraviesa documentacion
span El elemento de extension de contenido,es un contenedor en línea genérico para redactar contenido, que no representa inherentemente nada. Se puede usar para agrupar elementos con fines de estilo (usando los atributos o ), o porque comparten valores de atributo documentacion
strong El elemento de gran importancia,ndica que sus contenidos tienen una gran importancia, seriedad o urgencia documentacion
sub El elemento subindice,especifica texto en línea que debe mostrarse como subíndice únicamente por motivos tipográficos documentacion
sup El elemento superindice, especifica el texto en línea que se mostrará como superíndice únicamente por motivos tipográficos documentacion
time El elemento de (Fecha) Hora,representa un período específico en el tiempo. Puede incluir el atributo para traducir las fechas a un formato legible por máquina documentacion
u el elemento de anotación no articulada (subrayado),representa un tramo de texto en línea que debe representarse de manera que indique que tiene una anotación no textual documentacion
var El elemento variable,representa el nombre de una variable en una expresión matemática o un contexto de programación documentacion
video El elemento de insercion de video,incorpora un reproductor multimedia que admite la reproducción de video en el documento. También puede usarlo para contenido de audio documentacion
